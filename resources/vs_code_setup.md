# VS Code Setup for Java

Steps I followed to set up Java in VS Code:

1. Installed **Visual Studio Code**
2. Installed **Java Extension Pack** (includes Language Support, Debugger, Test Runner)
3. Installed **Code Runner** for quick execution
4. Installed **JDK (Java Development Kit)** on my system
5. Configured VS Code to:
   - Run `.java` files directly
   - Debug programs with breakpoints
   - Use integrated terminal for `javac` and `java` commands

This file will grow as I discover new extensions or settings.
