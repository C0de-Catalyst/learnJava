# 💻 Writing Your First Program

## Example: Hello World
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
```
## ✅ Output
```java
Hello, Java!
```

## 🔑 Key Concepts
Class & File Name: The file name must match the public class name ``` (HelloWorld.java)```.

Main Method: Every Java program starts with `public static void main(String[] args)`.

`System.out.println()`: Prints text to the console.

Compilation: Use `javac HelloWorld.java` to compile.

Execution: Use `java HelloWorld` to run
