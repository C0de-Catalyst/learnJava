# 🚀 Getting Started with Java

## 📖 Introduction
Java is a versatile, object-oriented programming language widely used for building applications ranging from desktop software to enterprise systems.  
Before writing code, you need to set up your environment properly.

---

## ⚙️ Setup Steps
1. **Install JDK (Java Development Kit)**  
   - Download from [Oracle JDK](https://www.oracle.com/java/technologies/javase-downloads.html) or [OpenJDK](https://openjdk.org/).
   - Add JDK `bin` folder to your system PATH.

2. **Install an IDE**  
   - Recommended: **Visual Studio Code** with Java Extension Pack.  
   - Alternatives: IntelliJ IDEA, Eclipse, NetBeans.

3. **Verify Installation**  
   Run in terminal:
   ```bash
   java -version
   javac -version
